Detailed Description

This is a struct representing a half-edge in the DCEL. A half-edge is a directed edge that forms a boundary between two regions in a planar subdivision.

Parameters
twinEach half-edge is uniquely identified by its twin half edge which is used to navigate the edges of the subdivision.
nextEach half-edge is uniquely identified by its next half edge which is used to navigate the edges of the subdivision.
prevEach half-edge is uniquely identified by its previous half-edges which is used to navigate the edges of the subdivision.
incident_vertexis a pointer to the vertex at the start of the half-edge.
incident_faceis a pointer to the face that the half-edge borders.
